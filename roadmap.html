Roadmap

This roadmap gives an overview over the project direction. A detailed list of features planned or under consideration can be found in the project boards; priorities are tracked by milestones:

  • Version numbers (0.1, 0.2, 0.2.1, …) track production releases. The next upcoming version has a target date.
  • backlog holds low-priority items.
  • unplanned holds zero-priority items (no feasible path to completion).

Note that all of these are tentative and can be changed, moved, or dropped at any time.

Next
