Neovim is vim out of the box.

Features

More powerful plugins

The msgpack API enables structured communication to and from any programming language. Remote plugins run as co-processes that communicate with Neovim safely and asynchronously.

Better out-of-the-box

  • Strong defaults
  • Modern terminal features such as truecolor and bracketed paste
  • Built-in terminal emulator

First-class embedding

GUIs (or TUIsâ€”such as readline) can nvim --embed or communicate via TCP socket using the discoverable msgpack API.

Drop-in replacement for Vim

Neovim is an extension of Vim: feature-parity and backwards compatibility are high priorities. If you are already familiar with Vim, see :help vim-differences to learn about the differences.

FAQs

Is it ready to download and run now with all the features?
The project is still undergoing frequent changes, but many people use it for their day-to-day work. Install instructions are available if you want to try it out.
When will a beta be available?
We are working hard to reach the 0.1 milestone. Take a look if you want to help out ;)
Will Neovim improve upon the default configuration that was found in Vim?
Yes.
Will Neovim deprecate Vimscript?
No.
